# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-21

### Added
- Initial release of Fitra Rizky Portfolio website
- Modern, responsive design inspired by Timmy O'Mahony's portfolio
- Next.js 16 with App Router
- TypeScript support
- Tailwind CSS for styling
- Framer Motion animations
- Dark mode support (system preference based)

#### Components
- **Header**: Responsive navigation with mobile menu
- **Hero**: Animated hero section with social links
- **Projects**: Featured projects grid with tags and hover effects
- **Photos**: Photo gallery with colorful placeholders
- **About**: Skills showcase and technology stack
- **Blog**: Latest blog posts with metadata
- **Contact**: Contact form and information
- **Footer**: Site footer with links and copyright

#### Features
- Smooth scroll animations
- Staggered entrance effects
- Hover interactions
- Mobile-first responsive design
- SEO optimization with meta tags
- Open Graph and Twitter Card support
- Custom scrollbar styling
- Accessibility considerations

#### Documentation
- Comprehensive README with setup instructions
- Contributing guidelines
- Deployment guide (Vercel, Netlify, AWS, Docker, VPS)
- MIT License
- Prettier configuration
- Environment variables template

### Technical Stack
- **Framework**: Next.js 16.0.3
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Package Manager**: npm

### Performance
- Optimized bundle size
- Fast page loads
- Smooth 60fps animations
- Lazy loading components
- Code splitting

### Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers
